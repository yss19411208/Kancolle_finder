# 艦これ ドロップ検索 ⚓

艦隊これくしょんの艦娘ドロップ海域を検索できるWebアプリケーションです。

## 特徴

- **2つの検索モード** — 海域から探す / 艦名から探す
- **艦種フィルター** — 艦種セレクトで素早く絞り込み
- **改造情報表示** — 改造先艦名と必要レベルを一覧表示
- **建造レシピ表示** — レシピ・確率・試行数をソート可能なテーブルで表示
- **海域攻略ページ** — 攻略ボタンからzekamashi.netの攻略ページをアプリ内で表示
- **レスポンシブ対応** — スマホでも快適に使用可能

## 使い方

### 海域から探す

1. 海域セレクトから海域を選択
2. ノードごとのドロップ艦一覧が表示される
3. 艦名をクリックすると詳細モーダルが開く

### 艦名から探す

1. 艦種セレクトで絞り込み（任意）
2. 艦名セレクトで艦娘を選択
3. ドロップ海域・建造レシピがタブ切り替えで表示される

## データ形式

### cancolle_data3.json

```json
{
  "長門": {
    "map": {
      "2-5": ["O"],
      "3-3": ["M"]
    },
    "type": "低速戦艦",
    "number": "80",
    "modify": {
      "長門改": "30",
      "長門改二": "88"
    }
  }
}
```

| キー | 内容 |
|------|------|
| `map` | ドロップ海域。`海域ID: [ノード, ...]` の形式 |
| `type` | 艦種 |
| `number` | 図鑑番号 |
| `modify` | 改造先艦名と必要レベル |

### build_data.json

```json
{
  "長門": {
    "build": [
      {
        "recipe": [400, 300, 600, 300],
        "Dev": 3,
        "count": 1500,
        "rate": 2.5,
        "big": false,
        "ship": ["陸奥", "扶桑", "...]
      }
    ]
  }
}
```

## 技術スタック
